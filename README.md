# Bank-Customer-segmentation
This project analyzes bank customer transaction data how Power BI can transform raw banking data into meaningful insights to understand customer behavior, identify high-value customers, reduce churn, risk levels, and revenue contribution, banks can improve personalization, enhance customer experience, and support data-driven decision-making 


## Business Objectives
- Understand customer behavior and transaction patterns
- Segment customers into New, Loyal, and Lost
- Apply RFM analysis to identify valuable customers
- Analyze customer risk using credit score
- Improve marketing and retention strategies

## Key Insights
- Loyal customers generate higher value per customer
- High churn risk observed in lost customer segment
- RFM analysis helps identify high-value customers
- High-risk customers require targeted monitoring

##  Tools Used
- Power BI
- DAX
- Power Query
- Excel / CSV
